CODSOFT Python Programming Internship
This repository contains the tasks completed as part of the CodSoft Python Programming Internship.

Tasks Completed

Task 1: To-Do List
->A command-line application to create, view, update, and delete tasks.
->Add new tasks
->View all tasks with completion status
->Mark tasks as complete
->Delete tasks

Run it:
python task1_todo.py

Task 2: Calculator
->A simple calculator that performs basic arithmetic operations.
->Takes two numbers and an operator as input
->Supports addition, subtraction, multiplication, and division
->Handles division by zero and invalid operations

Run it:
python task2_calculator.py

Task 4: Rock-Paper-Scissors Game
->An interactive game where the user plays against the computer.
->Random computer choice
->Determines winner based on game rules
->Tracks score across multiple rounds
->Option to play again

Run it:
python task4_rps.py

Tech Stack
Python 3
